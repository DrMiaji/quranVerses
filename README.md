## How To Use
Add This In root `build.gradle`
`allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}`
 Then add this in module `build.gradle`
  `dependencies {
	        implementation 'com.github.mosamabinomar:quranVerses:1.0'
	}`
